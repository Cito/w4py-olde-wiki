I'm working on a document which compares Python Web modules and frameworks to Java 2 Enterprise Edition frameworks. I believe it would be good to strengthen Webware's capability as a generic application server, but not necessarily by just copying from J2EE - there should be benefits in using Webware instead of J2EE application servers, just as there are benefits in using Python instead of Java for certain kinds of software development.
